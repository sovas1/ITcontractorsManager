IT Contractors Manager



Company Controller

@GET /company -- return list of company

@GET /company/{id} -- return single company

@POST /company -- add a company (json body required)

@PUT /company -- edit company (json body required)

@DELETE /company{id} -- delete company

@PUT /company/{companyId}/contractor/{contractorId} -- add contractor to company



Contractor Controller

@GET /contractor -- return list of contractor

@GET /contractor/{id} -- return single contractor

@POST /contractor -- add a contractor (json body required)

@PUT /contractor -- edit contractor (json body required)

@DELETE /contractor{id} -- delete contractor
